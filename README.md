# Nonlinear Electromagnet Simulation

## Overview
This project focuses on the simulation and analysis of a nonlinear electromagnet system using MATLAB and Simulink. The system models position-dependent inductance as the source of nonlinearity and evaluates the dynamic behavior of the electromagnet through simulation and engineering analysis.

## Features
- Nonlinear electromagnet modeling
- MATLAB-based engineering analysis
- Simulink dynamic simulation
- Position-dependent inductance analysis
- System response evaluation

## Tools Used
- MATLAB
- Simulink

## Files Included
- MATLAB source files (.m)
- Simulink models (.slx)
- Project report PDF
- Simulation output and image files
